# iRealPro
[戻る](./README.md)  
まずはライブラリーにある曲を再生してみると、各記号の使い方の参考になります。  
消したいときは、まずバックアップをしたうえで、「設定 -> ユーティリティー -> すべてのコンテンツと設定を消去する」で消すと、ライブラリがカラになります。  
バックアップだけを戻せば、すっきりします。  
削除した「練習曲」を復元するには、「設定 -> ユーティリティー -> 練習曲を復元する」です。
